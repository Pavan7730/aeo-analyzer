Privacy Policy

AEO Analyzer

Your privacy is important to us. This Privacy Policy explains how the AEO Analyzer Chrome Extension handles user data.

1. Data Collection

AEO Analyzer does not collect, store, transmit, or sell any personal data.

All analysis is performed locally within your browser on the currently opened webpage.

2. Website Content Analysis

The extension temporarily reads the visible content of the active webpage only when you click the “Analyze Page” button.

This content is used solely to calculate an Answer Engine Optimization (AEO) score and generate feedback. The content is never stored or shared.

3. No External Servers

AEO Analyzer does not send data to any external servers or third-party services. All processing happens entirely on your device.

4. Permissions

The extension uses the following permissions:

  • activeTab – to analyze the current webpage when requested.
  • scripting – to run the analysis logic on the page.

These permissions are used strictly for the extension’s core functionality.

5. Cookies & Tracking

AEO Analyzer does not use cookies, trackers, analytics tools, or advertising scripts.

6. Third-Party Access

No third parties have access to any data through this extension.

7. Changes to This Policy

This Privacy Policy may be updated if the extension’s functionality changes. Any updates will be reflected on this page.

8. Contact

If you have questions about this Privacy Policy, you may contact the developer via the GitHub repository associated with this extension.
